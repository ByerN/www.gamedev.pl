# Gamedev.pl

Gamedev.pl is a static website generated from various sources.

# Roadmap

# Development

Requirements:

- nodejs
- npm
- gulp

How to run:

    git clone git@github.com:gamedevpl/www.gamedev.pl.git
    cd www.gamedev.pl
    npm install
    gulp serve
